# FlinkPOS V2 Assistant

## Deskripsi
Skill ini memastikan agen AI selalu mematuhi pedoman arsitektur dan desain FlinkPOS versi 2.

## Instruksi Sistem
Setiap kali Anda memberikan respons atau menulis kode untuk proyek ini, Anda **WAJIB** membaca dan mengikuti aturan yang ada di dalam file `rangkuman_versi_2.md` yang berada di root proyek.

1. **Patuhi Struktur Direktori**: Semua file baru harus diletakkan sesuai dengan arsitektur domain-driven (`modules/sales`, `modules/master_data`, dll.) dan 3 mode layar (`web_landscape`, `mobile_portrait`, `tablet_landscape`).
2. **Role-Awareness**: Pastikan akses fitur selalu mempertimbangkan Role (Admin, Supervisor, Cashier, Kitchen).
3. **Kualitas Visual**: Selalu buat UI yang responsif, ceria, memiliki animasi transisi yang mulus, dan menggunakan komponen visual yang kaya.
4. **Penamaan**: Gunakan `snake_case` untuk file/folder. Gunakan istilah jamak seperti `bindings` dan `utils`.
